# Major Arcana Hero Artwork

Place one generated `.webp` artwork for each Major Arcana card in this folder.

Required filenames:

- `the-fool.webp`
- `the-magician.webp`
- `the-high-priestess.webp`
- `the-empress.webp`
- `the-emperor.webp`
- `the-hierophant.webp`
- `the-lovers.webp`
- `the-chariot.webp`
- `strength.webp`
- `the-hermit.webp`
- `wheel-of-fortune.webp`
- `justice.webp`
- `the-hanged-man.webp`
- `death.webp`
- `temperance.webp`
- `the-devil.webp`
- `the-tower.webp`
- `the-star.webp`
- `the-moon.webp`
- `the-sun.webp`
- `judgement.webp`
- `the-world.webp`

Artwork guidelines:

- No text, letters, numbers, typography, logos, or watermarks inside the image.
- Leave safe space for the app's HTML/CSS title overlay.
- Use premium mystical oracle card backgrounds with dark cosmic gradients.
- Keep the mood elegant, spiritual, gentle, and not scary.
- The app applies aura-color glow and overlays dynamically, so one image per card can support many aura moods.
